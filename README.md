# Health-Data-Analysis
Data analysis

## Introduction
This project analyzes basic health data of 20 people using Python.  
The dataset includes name, age, weight, and daily steps.

The program performs simple analysis such as:
- Average weight calculation
- Identifying the person with highest steps
- Classifying people as Active or Less Active
- Visualizing steps using a bar chart

## Technologies Used
- Python
- Matplotlib

## Dataset Fields
- Name
- Age
- Weight (kg)
- Steps per day

## Project Output
- Average Weight
- Most Active Person
- Activity Report
- Bar Graph of Daily Steps
